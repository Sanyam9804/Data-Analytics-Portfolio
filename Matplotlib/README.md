# 📈 Data Visualization (Matplotlib & Seaborn)

This folder contains my learning of data visualization using Matplotlib and Seaborn.

---

## 📚 Topics Covered

* Line Plots
* Bar Charts
* Histograms
* Scatter Plots
* Pie Charts
* Subplots
* Customization (labels, titles, colors)
* Seaborn Visualizations (heatmap, pairplot, etc.)

---

## 📂 Files

* line_plot.py → Basic line graphs
* bar_chart.py → Bar chart visualization
* histogram.py → Distribution of data
* scatter_plot.py → Relationship between variables
* pie_chart.py → Pie chart visualization
* seaborn_plots.py → Advanced visualizations using Seaborn

---

## 🎯 Goal

To create clear, meaningful, and visually appealing charts for data analysis and storytelling.
